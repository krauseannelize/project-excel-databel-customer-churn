## Project Overview

This project was developed as part of DataCamp’s *Fundamental Skill Track* case study “Analyzing Customer Churn in Excel.” It enhances the suggested solution by focusing on business‑relevant metrics and advanced visualizations. The outcome is a dynamic dashboard that provides actionable insights into churn drivers, demographics, and account tenure.

## Objectives

- Build an interactive dashboard to analyze customer churn for Databel  
- Identify key churn drivers and demographic trends  
- Provide business‑relevant insights for decision‑making  
- Showcase advanced Excel formulas and dynamic visualization techniques  

## Methodology

- Data preparation and cleaning in Excel  
- Construction of key metrics widgets (customers, churned, churn rate)  
- Visualization of churn reasons, age groups, demographics, and account tenure  
- Dynamic filtering and interactive map for state‑level churn analysis  
- Application of advanced Excel functions (LET, SWITCH, FILTER, PIVOTBY)  

## Key Findings

- **Churn Rate:** 26.86% overall, with 1,796 customers churned out of 6,687  
- **Churn Drivers:** “Competitor” accounts for 44.82% of churn, followed by “Attitude” at 15.98%  
- **Demographics:** Churn rate exceeds 40% for customers aged 79–88; highest churn among seniors and short‑tenure accounts  
- **Geography:** California shows a significantly higher churn rate (63.24%) compared to other states
